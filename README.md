# job-portal-frontend

Job Portal Frontend built with React 19 + Vite + Tailwind CSS + Spring Boot Backend Integration

## Features
- Modern React 19 with Vite
- Tailwind CSS for styling
- Axios for API integration
- React Router for navigation
- Context API for authentication
- Fully connected with Spring Boot Job Portal Backend

## Tech Stack
- React 19
- Vite
- Tailwind CSS
- Axios
- React Router DOM
- React Hook Form

## How to Run

```bash
npm install
npm run dev
